  - Key stored in code: `scripts/submit-indexnow.ts` (hardcoded as `b8f4e2a1c9d7e3f5`)
  - Auth: API key in request body
  - Host: `divkix.me`
  - Triggered during: Production builds (Cloudflare Pages main branch)

**Email & Form Submission:**
- Formspree - Form backend service
  - Endpoint: `https://formspree.io/f/xgvreprq`

  - Method: HTTP POST to form-specific endpoint
  - No explicit auth token (endpoint ID is form-specific UUID)
  - Receives: `name`, `email`, `message` fields as JSON
  - Returns: Success (200) or error response
